B1 - Khởi tạo Git: git init

B2 - Theo dõi Git: git add .

B3 - Commit: git commit -m "first commit"

B4 - Chuyển sang main: git branch -M main

B5 - Thêm vào lnik Git: git remote add origin https://github.com/NgocHoang02/duan3.git

B6 - Push code: git push -u origin main
